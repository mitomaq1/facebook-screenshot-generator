# Facebook Screenshot Generator

A web-based application for creating realistic Facebook screenshots with customizable profiles, posts, and timeline views.

## Features

- **Profile Customization**
  - Upload profile picture and cover photo
  - Edit name, bio, location, work, education
  - Customize relationship status, birthday, website
  - Real-time preview

- **Post Creation**
  - Create multiple posts with text content
  - Upload images to posts
  - Customize likes, comments, and shares count
  - Edit timestamps
  - Edit and delete existing posts

- **Authentic Facebook UI**
  - Realistic Facebook design (2025 style)
  - Profile header with cover photo
  - Profile picture overlay
  - Post cards with reactions/comments
  - Timeline layout

- **Export Functionality**
  - Capture entire profile view as high-quality image
  - Download as PNG
  - Custom filename

## Installation

1. Install dependencies:
```bash
npm install
```

2. Start development server:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
```

## Usage

1. Use the left sidebar to edit your profile information
2. Upload profile and cover photos
3. Create posts with text and images
4. Customize likes, comments, and shares
5. Click "Export Screenshot" to download your Facebook screenshot

## Technology Stack

- React + TypeScript
- Vite
- Tailwind CSS
- Zustand (state management)
- html2canvas (screenshot export)
- Lucide React (icons)
